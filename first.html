WHAT IS HTML?

Introduction to HTML

* What is HTML?

HyperText Markup Language.

* A Little History:

Early websites didn't even have search functionality, so users had to navigate manually using hyperlinks .

* The Need for Tags:

The raw text doesn't format correctly on a web page, which is why tags are needed to add structure and meaning.

Heading Tags (h1 to h6):

* Introduction:

The (h1) tag is introduced for creating a primary heading .

* Types of Headings:

All six heading tags (h1 to h6)

* Semantic Meaning vs Size:

The heading tags convey importance rather than just text size, which can be controlled with CSS

* SEO Benefits:

You'll learn how headings help with Search Engine Optimization (SEO) by providing structure for search engines .

* Single H1:

It's recommended to have only one (h1) tag per page .

* Avoid Jumping Headings:

The importance of maintaining a sequential order (h1 then h2, not H1 directly to h3).

Paragraph Tag (p):

* Purpose:

This tag is used for general text content .

* Lorem Ipsum:

The lorem shortcut in VS Code is introduced for generating placeholder text .

HTML Elements vs. Tags:

The difference between an HTML tag (e.g., p ,h ) and an HTML element (the opening tag, content, and closing tag).

Horizontal Rule Tag (hr):

* Purpose:

This tag is used to create a horizontal line on the web page .

* Self-Closing Tag:

The hr tag is a self-closing tag .

Break Tag (br):

* Purpose:

This tag is used to add a line break .

* Self-Closing Tag:

br is also a self-closing tag .

List Tags (ol, ul, li):

* Ordered List (ol):

This tag is used for numbered lists.

* Unordered List (ul):

This tag is used for bullet-point lists.

* List Item (li):

This tag is used for individual items in both ordered and unordered lists .

* Comments:

You'll learn how to add comments in HTML ( !-- --) to make your code more readable. Comments are not visible in the output .

Anchor Tag (a):

* Purpose:

This tag is used to create hyperlinks .

* (href="") Attribute:

The href attribute is used to specify the URL of the linked page .

* target Attribute (_blank):

The target="_blank" attribute is introduced to open links in a new tab.

* Attributes:

(e.g., href, target) properties that describe HTML tags .

Image Tag (img):

* Purpose:

This tag is used to embed images into a web page.

* (src="") Attribute:

The src attribute is used to specify the image source (URL or file path).

* Resizing Images:

The height and width attributes are used to control the image dimensions .

* (alt="") Attribute

The alt (alternative text) attribute is explained for providing descriptive text if the image doesn't load, which also helps with accessibility for screen readers .
