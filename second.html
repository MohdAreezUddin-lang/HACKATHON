Introduction to Nested Lists

The nested lists, which are lists created within other lists.

* Unordered Nested Lists:

How to create a basic unordered list and then nests another unordered list within it.
For example, a list of countries with a nested list of cities for each country.

* Ordered and Unordered Nested Lists:

How to nest an ordered list within an unordered list.
For instance, an unordered list of cities with a nested ordered list of areas within each city.

Introduction to Tables

* Basic Table Creation:

The fundamental HTML tags for creating a table: (table), (tr) (table row), and (td) (table data). The border attribute is also introduced to make the table borders visible.

* Using (th) for Table Headers:

The (th) (table header) tag is introduced as a way to define header cells in a table, improving the table's structure and accessibility.

* Professional Table Structure:

Using (thead) refers to (table head), (tbody) refers to (table body), and (tfoot) refers to (table foot) tags.

* Using (colspan=""):

The (colspan="") attribute is explained, which allows a single table cell to span the width of multiple columns.

* Using (rowspan=""):

The (rowspan="") attribute is introduced, which allows a single table cell to span the height of multiple rows.

* Complex Table with rowspan and colspan:

That uses both rowspan and colspan to create an intricate layout with merged cells.

For example:

Countries India Russia China
Population 1.466B 144M 1.415B
