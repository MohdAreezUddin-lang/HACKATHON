Introduction to the Internet

This section delves into the core components and workings of the internet

* Purpose of the Internet

The internet is explained as a global network for communication and data exchange between various devices.

* IP Address(Internet protocol)

An IP(Internet protocol) address is defined as a unique identifier for devices on the internet, same as a physical address for mail delivery.

* Internet Service Provider (ISP):

The role of an ISP is to assign temporary IP addresses to devices upon connection to the internet.

* DNS (Domain Name System):

DNS servers translate human-friendly domain names (e.g., "google.com") into machine-readable IP addresses.

* MAC Address(Media Access Control Address):

The MAC address is introduced as a permanent, physical address for a device, which is essential for secure data transmission.

Single device can have multiple MAC addresses.

* Port Number:

The function of port numbers in directing data to the correct application or tab within a device. There are different port numbers.

* HTTP vs HTTPS:

The distinction between the insecure HTTP and the secure, encrypted HTTPS protocols are refered for data transfer.

Networking Fundamentals

* Data Transfer Methods:

The evolution of data transfer methods is starting from basic solutions like USB drives and direct cabling to more scalable networking solutions.

* Switches and Local Area Network (LAN):

How switches are used to create local networks, enabling communication between devices within a limited area.

* Routers:

The function of routers in connecting different local networks to form the global internet. The concept of routers incorporating switches and Wireless Access Control (WAC).

* Submarine Cables:

The role of submarine cables is to connect continents.

* Public vs Private IP Addresses:

The difference between globally unique public IP addresses and locally unique private IP addresses, and the reasons forthe introduction of private IP addresses.
